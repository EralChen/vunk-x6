## 技术设计

大模型节点字段

| 字段名 | 类型 | 描述 |
| --- | --- | --- |
| modelId | string | 基础模型ID |
| output | Field[] | 输出 |
| input | InputField[] | 输入 |
| prompt | string | 提示词 |

```ts
interface Field {
  name: string
  type: string
  description?: string
}

interface ValueRef {
  id: string
  name: string
}

interface InputField extends Field {
  // 值引用
  valueRef: ValueRef
}
```
